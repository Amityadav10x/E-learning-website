    <h1>EduConnect: Multi-Page Educational Website</h1>

<h2>Project Overview</h2>
    <p>EduConnect is a responsive multi-page website designed to provide a seamless and engaging educational experience. The platform includes essential features like user authentication, course information, and contact options, offering a user-friendly interface for learners and educators.</p>

 <h2>Features</h2>
    <ul>
        <li><strong>Login Page</strong>: Secure user authentication.</li>
        <li><strong>Home Page</strong>: Overview of the platform with key highlights.</li>
        <li><strong>About Page</strong>: Details about the platform's mission and vision.</li>
        <li><strong>Courses Page</strong>: Comprehensive list of available courses.</li>
        <li><strong>Contact Page</strong>: Form to connect with the team for inquiries or support.</li>
    </ul>

  <h2>Technologies Used</h2>
    <ul>
        <li><strong>Frontend</strong>: HTML, CSS, JavaScript, jQuery.</li>
        <li><strong>Design</strong>: Responsive layouts ensuring compatibility across devices.</li>
        <li><strong>Interactivity</strong>: Dynamic features for an enhanced user experience.</li>
    </ul>

  <h2>Installation and Setup</h2>
    <ol>
        <li>Clone the repository:
            <pre><code>git clone https://github.com/your-repo/educonnect.git</code></pre>
        </li>
        <li>Navigate to the project folder:
            <pre><code>cd educonnect</code></pre>
        </li>
        <li>Open <code>index.html</code> in your browser to view the website.</li>
    </ol>
